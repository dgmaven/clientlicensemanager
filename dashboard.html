1. Validating a License (Incoming API)

POST {{ configUrl || 'https://script.google.com/.../exec' }}

Headers:
Content-Type: application/json

Body:
{
  "action": "validateLicense",
  "payload": {
    "apiKey": "{{ settingsData.APISecret || 'YOUR_API_SECRET' }}",
    "licenseKey": "ABCD-1234-WXYZ-5678"
  }
}

Example API Response

{
  "status": "success",

  }
}
